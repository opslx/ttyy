## 测试请使用以下命令：
```
git clone https://github.com/opslx/ttyy.git
cd tttyy
npm install pnpm
npm install vite
pnpm install
npx vite --host=0.0.0.0 --port=4000
```
访问本地4000端口。
编译请使用```npx vite build```
